# Digital Avatar 使用示例

## 基本使用

### 启动数字人

```bash
nanobot
```

数字人会自动启动，并在浏览器中打开 `http://127.0.0.1:18791`

## 自定义表情触发

### 触发 happy 表情
```

"好的！"
```

### 触发 thinking 表情


### 触发 surprised 表情

```
"天哪！"
"真的吗？"
```

### 触发 confused 表情


## 注意事项

1. **表情切换是自动的**: 数字人会根据对话内容和上下文自动切换表情
2. **图片是随机选择的**: 同一类型的图片会随机选择，增加多样性
3. **表情检测基于关键词**: 确保你的对话包含相关的关键词
4. **Web 界面实时更新**: 数字人的表情和状态会在 Web 界面实时更新

## 高级用法

### 自定义图片

你可以在 `assets/` 目录中添加自己的图片：

```bash
# 添加正面照
cp my-avatar-neutral.png /path/to/digital-avatar/assets/正面照-1.png

# 添加说话状态
cp my-avatar-speaking.png /path/to/digital-avatar/assets/说话-1.png

# 添加表情
cp my-avatar-happy.png /path/to/digital-avatar/assets/表情-开心.png
```

### 自定义表情关键词

编辑 `scripts/expression_manager.py` 中的 `KEYWORD_MAP`：

```python
KEYWORD_MAP = {
    "happy": Expression.HAPPY,
    "开心": Expression.HAPPY,
    "成功": Expression.HAPPY,
    # 添加你自己的关键词
    "太好了": Expression.HAPPY,
    "完美": Expression.HAPPY,
}
```

## 故障排除

### 表情未切换

1. 检查对话内容是否包含相关关键词
2. 检查 `expression_manager.py` 中的关键词映射
3. 查看日志输出，确认表情检测结果

### 图片未加载

1. 检查 `assets/` 目录是否存在图片
2. 检查图片文件名是否包含正确的关键词
3. 查看日志输出，确认图片加载情况

### Web 界面未更新

1. 检查浏览器是否正常连接到 `http://127.0.0.1:18791`
2. 检查端口是否被占用
3. 尝试刷新浏览器页面
